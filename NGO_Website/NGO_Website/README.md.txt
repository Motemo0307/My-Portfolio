Daily Bread Foundation – Website

A clean, responsive website built for the Daily Bread Foundation, a non-profit organization based in Tembisa, South Africa. The website showcases the foundation’s mission, services, events, and community support programs.

Purpose of the Project
This website was created to help the organization:
•	Present their work professionally
•	Share photos of community projects
•	Display sponsors and services
•	Allow visitors to easily donate or contact the organization
•	Build trust with the community and future partners

     Features
•	Hero banner with mission statement
•	Auto-sliding image gallery with short messages
•	Services section describing programs
•	Sponsors page
•	Image gallery with lightbox preview
•	Responsive design (works on mobile & desktop)
•	Contact page with email and phone info

    Technologies Used
•	HTML5
•	CSS3
•	JavaScript 
•	Font Awesome Icons

    Image Slider (Home Page)
The homepage contains a 4-image slider that automatically changes every few seconds.
Each image displays a short inspiring message related to charity and community support.
